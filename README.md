<strong>Backend</strong>

<ul>
<li> Restfull Api for CRUD operation </li>
<li> Swagger api documentation </li>
<li> Jwt authentication </li>
<li> image upload multer </li>
</ul>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<ul>

<li> Rect query </li>

</ul>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run build

# watch mode
$ npm run dev

# production mode
$npm start
```
