<strong>Backend</strong>

<ul>
<li> Restfull Api for CRUD operation </li>
<li> Swagger api documentation </li>
<li> Jwt authentication </li>
<li> image upload multer </li>
</ul>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
